### 📝 학습 전략

#### 1단계: 구현 문제 완벽 마스터 (1-2주)
- **목표**: 30분 내에 구현 문제 100% 정확도로 해결
- **일일 학습량**: 2-3문제
- **방법**: 
  - 기본 구현 7문제 반복 연습 (3일)
  - 문자열 문제 6문제 집중 (2일)
  - 수학/시뮬레이션 7문제 (2일)
  - 시간 단축 연습 (처음 1시간 → 목표 20-30분)
  - **체크포인트**: 1주차 말 모의 테스트로 20분 내 정확도 검증

#### 2단계: 알고리즘 기초 다지기 (2주)
- **목표**: DFS/BFS, 기초 DP 완벽 이해
- **일일 학습량**: 2문제
- **우선순위 1**: DFS/BFS 완전 정복 (1주)
  - 그래프 기초 5문제 (DFS/BFS 개념)
  - BFS 활용 6문제 (최단거리, 탐색)
  - **체크포인트**: BFS 문제 40분 내 해결
  
- **우선순위 2**: 기초 DP 패턴 학습 (1주)
  - 1차원 DP 6문제 (계단오르기, 타일링 등)
  - 2차원 DP 4문제 (LIS, RGB거리 등)
  - **체크포인트**: DP 점화식 10분 내 도출

#### 3단계: 백트래킹 집중 훈련 (2주)
- **목표**: 백트래킹 패턴 완벽 숙달 및 가지치기 최적화
- **일일 학습량**: 1-2문제 (고난도)
- **방법**:
  - N과 M 시리즈 5문제로 기본 패턴 (4일)
  - 조합/순열 응용 4문제 (4일)
  - 고난도 백트래킹 3문제 (3일)
  - **체크포인트**: 시간초과 없이 통과율 80% 이상

#### 4단계: 스택/그리디 최적화 (1주)
- **목표**: 시간복잡도 최적화 감각 익히기
- **방법**:
  - 스택 4문제 (오큰수 포함)
  - 그리디 5문제
  - **체크포인트**: O(N²) → O(N) 최적화 능력

#### 5단계: 시뮬레이션 고난도 문제 (2주)
- **목표**: 복잡한 구현 + 알고리즘 결합 문제 해결
- **일일 학습량**: 1문제 (구현 시간 고려)
- **방법**:
  - 삼성 SW 스타일 5문제
  - 디버깅 연습 (테스트케이스 직접 작성)
  - 코드 모듈화 및 리팩토링
  - **체크포인트**: 2시간 내 70% 이상 구현
